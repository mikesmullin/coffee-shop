## Introduction

Watch the 15min intro to [Installing Node.js CoffeeShop v0.1.1](http://youtu.be/sdVvesNOn6g) on YouTube.

## Install

```bash
sudo apt-get install nodejs
sudo npm install coffee-shop -g
shop new <project>
```
